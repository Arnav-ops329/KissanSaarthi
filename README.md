
# 🌾 Kissan Saarthi – Smart Farming AI App

Kissan Saarthi is an AI-powered smart farming assistant designed to help farmers make better decisions using technology. The app integrates **AI disease detection, real-time mandi prices, weather insights, fertilizer recommendations, and voice-based control**.

---

## 📱 Features

### 🌿 1. Crop Disease Detection

* Upload images from gallery
* AI-based disease prediction
* Instant results with crop & disease info

---

### 💰 2. Real-Time Mandi Prices

* View latest crop prices
* Avoid middlemen scams
* Identify best market to sell

---

### 🌦 3. Weather Forecast

* Real-time temperature, humidity, wind
* Location-based weather (coming soon)
* Smart farming tips based on weather

---

### 🧪 4. Fertilizer Calculator

* Input crop, soil, weather, and land area
* Get accurate NPK recommendations
* Cost estimation to save money

---

### 🌱 5. Crop Recommendation System

* Suggest best crops based on:

  * Weather
  * Soil type
  * Market demand
* Helps maximize profit

---

### 🎤 6. Voice Assistant 

* Fully voice-driven navigation
* Commands like:

  * "Scan crop"
  * "Open mandi prices"
* Supports regional languages (in progress)

---

### 🌍 7. Multi-Language Support

* Dynamic translation system
* Supports Indian languages
* Designed for rural accessibility

---

## 🧠 Tech Stack

### 📱 Frontend (Flutter)

* Flutter (Dart)
* Provider (State Management)
* Camera & Image Picker
* Speech-to-Text
* Text-to-Speech

### 🌐 Backend

* Node.js + Express
* API integration (Agri data, Weather)
* AI model integration (disease detection)

## ⚙️ Setup Instructions

### 🔹 1. Clone Repository

```
git clone https://github.com/your-username/kissan-saarthi.git
cd kissansaarthi
```

---

### 🔹 2. Install Dependencies

```
flutter pub get
```

---

### 🔹 3. Run Backend

```
cd backend
npm install
node server.js
```

---

### 🔹 4. Run App

```
flutter run
```
